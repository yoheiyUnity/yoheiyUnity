【0から始めるUnity】Unityをインストールしよう

 このドキュメントではUnityのインストールの仕方について解説しています。

 1.Unity Hubをインストールする

 2.Unity IDの作成とログイン

 3.Unityのバージョンを選んでインストール

 2023年12月11日時点の情報で書いていますが、可能な限りUnityのマニュアルを参照して書いていますので、時間が経っていても新しいバージョンのマニュアルを参照してインストールできます。

 問題ないようでしたら後の項目に進んでください。

1.Unity Hubをインストールする

 Unity HubはUnityのプロジェクトとUnityエディタのバージョンを管理するためのソフトウェアです。

 Unityはたびたび更新されており、バージョン番号が変わります。

 バージョンが変わるとプロジェクトの互換性がなくなり以前のプロジェクトが開けなくなったり、動作が変わってしまう場合がありますので作った時点と同じ、もしくはそれより後でバージョンの近いもので開く必要があります。

 そのため、複数のプロジェクトを作るとそれらを開くために別々なバージョンのUnityをインストールする必要が出てきます。

 それらを管理するために使われるのがUnity Hubです。

 Unity Hubのインストール方法についてはUnityのマニュアルを参照してください。

 https://docs.unity3d.com/ja/2021.2/Manual/GettingStartedInstallingHub.html

 マニュアルを参照する際は左上のバージョンが適切に選ばれているかどうかをチェックしてください。最新のバージョンに対応していなくてもできるだけ新しいものを参照しましょう。

2.Unity IDの作成とログイン

 Unity HubやUnity Editorを使うにはUnity IDが必要です。すでにお持ちの方はそのIDを使ってログインしてください。お持ちではない方は下記のURLからUnity IDを作成してください。

 https://id.unity.com/ja/login

 Unityには学生向け、個人向け、企業向けなど様々なライセンスがあり、一つのUnity IDに対して複数のライセンスを割り当て、切り替えて使用する事ができます。ライセンスの区分に関しては以下のサイトを参照してください。

 https://unity.com/ja/pricing#plans-student-and-hobbyist

 組織でUnityを使用するわけでなく、初めてUnityを使う場合は個人向けのPersonalを使うことになりますのでUnity Hub>Prefarence>Licensesの画面を開き、画面右上のAddボタンを押してAdd new license>Get a free personallicenseでPersonal Licenseを取得してください。

 Professionalなどのライセンスを組織から与えられた場合はActivate with serial numberを選びシリアル番号を入力してください。

3.Unityのバージョンを選んでインストールする

 Unity Hubにログインし、ライセンスの設定をしたらUnity Editorのインストールを行いましょう。Installsをクリックすると現在PCにインストールされているUnity Editor一覧が出てきます。初めてUnityをインストールする場合はもちろん何も出てきません。

 ここで右上のInstall EditorをクリックするとインストールするとUnity Editorのバージョンを選ぶ画面になります。基本的にはOfficial Releasesの中のLONG TERM SUPPORT(LTS)の中から新しいバージョンを選ぶのが良いのですが、チュートリアルのドキュメントやビデオを見ながら同じことをする、既存のUnityプロジェクトを開くなどの場合は、それらとバージョンを合わせる必要があります。
表示されたリストの中に目的のバージョンが見当たらない場合はArchiveからUnityのWebサイトに飛び、目的のバージョンを直接ダウンロードしてください。

 なおプロジェクトに使われたUnityのバージョンはUnity HubのProjectsから見ることができますので、バージョンがわからない場合はProjects画面からAddを押してプロジェクトのフォルダを選択しリストに加えてください。

 無事にインストールできたでしょうか？

 YoutubeにはUnityに関する最新情報やチュートリアルなどをお届けするUnity Japanのチャンネルもありますので、ぜひチャンネル登録をしてください！

 https://www.youtube.com/@unity_japan
